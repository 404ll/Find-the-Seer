# 🪄 Find-the-Seer

**[English](README.en.md)** | **中文**

### *这个世界没有真正的预言家 —— 只有群体的判断。*

## 🌍 项目概述

**Find-the-Seer** 是一个基于 **Sui** 和 **Walrus** 构建的去中心化“预言”互动平台，
这里 **没有真正的预言家**，一切的“真与假”都由群体的认知来决定。

用户可以发布“预言帖子”，内容可以是天气、事件、趋势甚至玩笑。
当时间到达设定的“兑现时刻”后，系统不会去验证结果真假，
而是根据 **最终投票比例与发帖人设定的预测比例的契合程度** 来分配奖励。

---

## 🧩 运作机制

1. **发布预言**

   * 用户（即“贴主”）发布一个带文字和图片的预言贴。
   * 帖子内容存储在 **Walrus** 去中心化存储中。
   * 发帖时需设定：

     * **兑现时间**（预言结束时间）
     * **真/假比例**（例如 70% 真 / 30% 假）
   * 发帖需要支付一笔 **SUI 手续费**。

2. **投票与质押**

   * 其他用户可在兑现时间 **前两天** 看到预言贴。
   * 他们根据自己的判断选择 “真” 或 “假”，并质押 **0.1 SUI**。

3. **兑现与奖励**

   * 当兑现时间到达时，系统公布最终投票比例。
   * 奖励分配逻辑如下：

     * **贴主**：预测比例越接近最终结果，奖励越高。
     * **投票者**：与最终比例方向一致的投票者获得奖励。
     * 投错方向的投票者会失去部分质押，作为奖励池分配给正确方。

4. **影响力机制**

   * 每个贴主的 **影响力值** 会根据其帖子总参与人数累积。
   * 高影响力贴主可获得来自平台资金池的额外奖励。

## 💰 平台经济模型

**平台盈利来源：**

* 每个预言帖的发布手续费（如 0.1 SUI）
* 每次结算时对质押奖池抽取 **2%–5% 平台手续费**
* 未来可开放 **高影响力NFT徽章或称号** 售卖

**奖励分配比例：**

* 70% → 玩家奖励（投票者 + 贴主）
* 25% → 平台可持续发展资金池
* 5% → 治理金库 / 未来生态激励

## 🎯 核心理念

> “没有真正的预言家，只有集体的判断。”

Find-the-Seer 将“预言”变成一种 **群体博弈与认知游戏**：
它不在于你是否知道真相，而在于你是否能 **预测群体将如何看待真相**。
